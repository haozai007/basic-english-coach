# Workplace Scenarios

Use this reference when the user asks for English phrases for a job, workplace task, meeting, email, interview, customer support, software work, sales, operations, or management.

## Output Format

For each phrase, prefer:

- Chinese
- Basic-English-like
- Natural simple English
- Note, if useful

## Meetings

- 我想确认一下重点。
  - Basic-English-like: I want to make certain about the main point.
  - Natural: I want to confirm the key point.
- 我们需要更多时间。
  - Basic-English-like: We need more time.
  - Natural: We need more time.
- 这个部分还不清楚。
  - Basic-English-like: This part is not clear.
  - Natural: This part is still unclear.
- 我会会后发给你。
  - Basic-English-like: I will send it to you after the meeting.
  - Natural: I will send it to you after the meeting.

## Email

- 请看附件。
  - Basic-English-like: Please see the file with this email.
  - Natural: Please see the attached file.
- 期待你的回复。
  - Basic-English-like: I am waiting for your answer.
  - Natural: I look forward to your reply.
- 如果你有问题，请告诉我。
  - Basic-English-like: If you have any questions, please let me know.
  - Natural: If you have any questions, please let me know.

## Software And Product Work

- 这个功能还没有完成。
  - Basic-English-like: This function is not ready yet.
  - Natural: This feature is not finished yet.
- 我发现了一个问题。
  - Basic-English-like: I found a problem.
  - Natural: I found an issue.
- 我会修复它。
  - Basic-English-like: I will make it right.
  - Natural: I will fix it.
- 这个需求需要确认。
  - Basic-English-like: We need to make certain about this need.
  - Natural: We need to confirm this requirement.

## Customer Support

- 很抱歉给你带来不便。
  - Basic-English-like: I am sorry for the trouble.
  - Natural: I am sorry for the inconvenience.
- 我会帮你查看。
  - Basic-English-like: I will look at this for you.
  - Natural: I will check this for you.
- 请给我更多信息。
  - Basic-English-like: Please give me more information.
  - Natural: Please provide more details.

## Sales

- 我想了解你的需求。
  - Basic-English-like: I want to know what you need.
  - Natural: I would like to understand your needs.
- 我们可以安排一次电话。
  - Basic-English-like: We can have a call.
  - Natural: We can schedule a call.
- 这个方案适合你的团队。
  - Basic-English-like: This plan is good for your team.
  - Natural: This solution is a good fit for your team.

## Interview

- 我负责这个项目。
  - Basic-English-like: I was the person working on this project.
  - Natural: I was responsible for this project.
- 我学得很快。
  - Basic-English-like: I get new knowledge quickly.
  - Natural: I learn quickly.
- 我想进一步提升英语沟通能力。
  - Basic-English-like: I want to make my English communication better.
  - Natural: I want to improve my English communication skills.
