# Translation Style

Use this reference for Chinese-to-English translation and simplification.

## Translation Steps

1. Understand the meaning in Chinese.
2. Identify the core action and actor.
3. Remove unnecessary abstract wording.
4. Split long sentences.
5. Create a Basic-English-like version.
6. Create a natural simple English version when useful.
7. Explain the most important word choices in Chinese.

## Preferred Output

```text
Basic-English-like:
...

Natural simple English:
...

Why:
- ...
- ...

Practice:
Try to say: ...
```

## Chinese Patterns

### 我想确认...

- Basic-English-like: I want to make certain ...
- Natural: I want to confirm ...

### 是否已经...

- Basic-English-like: if ... has already ...
- Natural: whether ... has already ...

### 麻烦你...

- Basic-English-like: Please ...
- Natural: Could you please ...

### 我这边...

Usually remove it unless it means "on my side".

### 需要推进...

- Basic-English-like: We need to move this forward.
- Natural: We need to move this forward.

### 对齐一下...

- Basic-English-like: We need to make certain we have the same idea.
- Natural: We need to align on this.

## Tone Choices

- Direct: Please send me the file.
- Polite: Could you please send me the file?
- Softer: When you have time, could you send me the file?
- Basic-English-like: Please send me the file when you have time.

## Avoid

- Do not translate Chinese word order mechanically.
- Do not overuse formal words when a simple word works.
- Do not hide useful natural English. Give both simple and natural versions when needed.
- Do not say a word is in the original Basic English list unless verified.
